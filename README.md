# Simulation Zoo

Whenever I want to learn a new programming language, I run into the same problem:

> "Okay, now what?"

The usual suggestions are calculators, TODO apps, REST APIs, or yet another command-line notes application. Those are fine (look at half of my projects), but I find them about as exciting as reading compiler error messages for entertainment.

So instead, I build simulations.

This repository is a collection (or perhaps containment facility) of small physical and mathematical simulations implemented in different programming languages as I learn them. The goal is not to produce the fastest, cleanest, or most scientifically rigorous implementation. The goal is to become familiar with a language by solving a problem I have already solved several times in the past.

Although these projects are primarily for my own learning, they may also serve as examples, references, or starting points for others exploring a new language.

Most implementations are intentionally self-contained and designed as one-off projects. In many cases I deliberately use language-specific features, patterns, and idioms, even when they make the code slightly more complicated than strictly necessary. If something looks suspiciously over-engineered for a toy simulation, that is probably because I was trying to learn something.
